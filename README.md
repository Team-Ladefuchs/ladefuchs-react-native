## How to install

```sh
npm install
npm i lottie-react-native
npm install @react-native-picker/picker --save
```

## Create .env file

```
API_TOKEN=12xxxxxxxxxxx

```

## How to run

iOS

```sh
npm run ios
```

Web

```sh
npm run web
```

Android

```sh
npm run android
```
